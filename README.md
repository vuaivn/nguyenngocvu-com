# nguyenngocvu.com

Website cá nhân của **Nguyễn Ngọc Vũ** — Blog + Portfolio, Tiếng Việt, xây bằng [Astro](https://astro.build).

> Tự do · Bình yên · Tạo giá trị bền vững

## Chuyên mục
- 🧘 **Phật pháp** — thiền, chánh niệm, ứng dụng Phật học vào đời sống
- 🤖 **Công nghệ & AI** — AI, tự động hóa, công cụ số
- 🌱 **Phát triển bản thân** — tự do, kỷ luật, tư duy phát triển

## Phát triển

```bash
npm install
npm run dev      # http://localhost:4321
npm run build    # xuất ra dist/
```

## Deploy
Xem [DEPLOY.md](./DEPLOY.md) — deploy tự động lên Cloudflare Pages khi push `main`.

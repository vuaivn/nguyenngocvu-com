Website cá nhân là nơi bạn kể câu chuyện của mình mà không phụ thuộc vào thuật toán mạng xã hội. Tin vui là: xây một cái vừa nhanh vừa miễn phí giờ đây rất dễ.

## Vì sao chọn Astro?

- **Nhanh:** xuất ra HTML tĩnh, tải gần như tức thì.
- **Chuẩn SEO:** hỗ trợ sitemap, RSS, meta tag gọn gàng.
- **Dễ viết nội dung:** viết blog bằng Markdown đơn giản.

## Vì sao chọn Cloudflare Pages?

- **Miễn phí** cho nhu cầu cá nhân.
- **Tự động deploy:** mỗi lần bạn push code lên GitHub, site tự cập nhật.
- **Nhanh toàn cầu** nhờ mạng CDN của Cloudflare.

## Quy trình tổng quát

1. Tạo dự án Astro trên máy.
2. Viết nội dung và tùy chỉnh giao diện.
3. Đẩy code lên GitHub.
4. Kết nối GitHub với Cloudflare Pages.
5. Trỏ tên miền — xong!

## Lời khuyên

Đừng chờ đến khi "hoàn hảo" mới ra mắt. Hãy bắt đầu với một trang giới thiệu đơn giản, rồi thêm dần. Chính website này (nguyenngocvu.com) cũng được xây theo cách đó.
